本项目收集了一些前端开发规范、流程、制度、sublime插件和一些常见问题。

##前端规范


###1.规范目的：
为提高团队协作效率,便于后台人员添加功能及前端后期优化维护,输出高质量的文档,统一标准，便于多人开发维护时代码统一，减少项目沟通和交接的成本，增加代码的可读性。
